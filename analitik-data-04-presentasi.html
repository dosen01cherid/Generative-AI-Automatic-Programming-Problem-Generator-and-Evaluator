üîç Apa itu Eksplorasi Data?

Definisi: Eksplorasi data adalah proses awal dalam analisis data yang bertujuan untuk memahami karakteristik, pola, dan struktur dataset sebelum melakukan analisis lebih lanjut.

  • Sebelum analisis formal

  • Sebelum pemodelan

üìã Aktivitas dalam Eksplorasi Data

1. Descriptive Statistics

Menghitung ukuran statistik dasar untuk memahami distribusi data:

  • Central Tendency: Mean, Median, Mode
  • Dispersion: Range, Variance, Standard Deviation
  • Distribution: Skewness, Kurtosis

Mean: \( \mu = \frac{1}{n}\sum_{i=1}^{n}x_i \)

Standard Deviation: \( \sigma = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(x_i - \mu)^2} \)

üìç Ukuran Pemusatan Data (Central Tendency)

Contoh Dataset: Nilai ujian 10 mahasiswa

75, 82, 90, 68, 88, 92, 78, 85, 88, 74

1. Mean (Rata-rata) = 82.0

\( \text{Mean} = \frac{75+82+90+68+88+92+78+85+88+74}{10} = \frac{820}{10} = 82 \)

‚úì Cocok untuk data normal, ‚úó Sensitif terhadap outlier

2. Median (Nilai Tengah) = 83.5

Data terurut: 68, 74, 75, 78, 82, 85, 88, 88, 90, 92

\( \text{Median} = \frac{82 + 85}{2} = 83.5 \)

‚úì Robust terhadap outlier, ‚úì Cocok untuk data skewed

3. Mode (Nilai Paling Sering) = 88

Nilai 88 muncul 2 kali (paling sering)

‚úì Cocok untuk data kategorikal, ‚úì Bisa ada lebih dari satu mode

üìè Ukuran Penyebaran Data (Dispersion)

Dataset yang sama: 75, 82, 90, 68, 88, 92, 78, 85, 88, 74 | Mean = 82

1. Range (Rentang)

\( \text{Range} = \text{Max} - \text{Min} = 92 - 68 = 24 \)

Menunjukkan perbedaan antara nilai tertinggi dan terendah

2. Variance (Varians)

\( s^2 = \frac{\sum(x_i - \bar{x})^2}{n-1} = \frac{(75-82)^2 + (82-82)^2 + ... + (74-82)^2}{9} \approx 64.4 \)

Rata-rata kuadrat deviasi dari mean

3. Standard Deviation (Simpangan Baku)

\( s = \sqrt{s^2} = \sqrt{64.4} \approx 8.03 \)

Dalam satuan yang sama dengan data asli - lebih mudah diinterpretasi

Interpretasi: 68% data berada dalam ¬±8.03 dari mean (74-90)

üì¶ Box Plot dan Deteksi Outlier

Komponen Box Plot:

  • Q1 (Kuartil 1): 25% data terkecil
  • Q2 (Median): 50% data
  • Q3 (Kuartil 3): 75% data
  • IQR: Q3 - Q1
  • Whiskers: 1.5 √ó IQR

Outlier jika:

\( x < Q1 - 1.5 \times IQR \)

\( x > Q3 + 1.5 \times IQR \)

Contoh: Gaji karyawan: Q1=4 juta, Q3=8 juta, IQR=4 juta

Outlier jika gaji < -2 juta atau > 14 juta

üéØ Outlier: Deteksi dan Penanganan

Contoh Dataset Penjualan Harian (dalam juta):

2.5, 2.8, 3.1, 2.9, 3.2, 2.7, 15.0, 3.0, 2.6, 3.3

Metode 1: Z-Score

\( Z = \frac{x - \mu}{\sigma} \)

Mean = 4.11 juta, SD = 3.74 juta

Z-score untuk 15.0 = (15.0 - 4.11) / 3.74 = 2.91

Kesimpulan: |Z| > 2.5 ‚Üí Outlier terdeteksi!

Metode 2: IQR Method

Q1 = 2.7, Q3 = 3.2, IQR = 0.5

Batas bawah = 2.7 - 1.5(0.5) = 1.95

Batas atas = 3.2 + 1.5(0.5) = 3.95

Kesimpulan: 15.0 > 3.95 ‚Üí Outlier!

Penanganan Outlier:

  • Investigasi: Cek apakah error data atau fenomena nyata
  • Removal: Hapus jika terbukti error
  • Transformation: Log transform untuk mengurangi dampak
  • Capping: Ganti dengan nilai batas (winsorization)

üîÑ Transformasi Data

Mengapa Transformasi?

  • Menormalkan distribusi skewed
  • Mengurangi pengaruh outlier
  • Memenuhi asumsi model statistik

1. Log Transformation

\( y = \log(x) \)

Contoh: Pendapatan ‚Üí Log(Pendapatan)

Data: 1000, 10000, 100000 ‚Üí Log: 3, 4, 5

2. Square Root Transformation

\( y = \sqrt{x} \)

Contoh: Jumlah Kasus ‚Üí ‚àöKasus

Data: 4, 16, 64, 256 ‚Üí ‚àö: 2, 4, 8, 16

3. Standardization (Z-score)

\( z = \frac{x - \mu}{\sigma} \)

Mengubah mean = 0, SD = 1

Data: 50, 60, 70 (Œº=60, œÉ=10) ‚Üí Z: -1, 0, 1

4. Min-Max Normalization

\( x' = \frac{x - x_{min}}{x_{max} - x_{min}} \)

Skala data ke rentang [0,1]

Data: 10, 50, 90 ‚Üí Normalized: 0, 0.5, 1
