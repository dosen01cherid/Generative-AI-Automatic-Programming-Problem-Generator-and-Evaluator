Tipe Data Analytics

CPMK 043
Sub-CPMK 1.3

Kemampuan Akhir yang Diharapkan

Mahasiswa mampu mendefinisikan data analytics & tipenya

4 Tipe Data Analytics

1. Descriptive Analytics (Apa yang terjadi?)

Menganalisis data historis untuk memahami apa yang telah terjadi

  • Contoh: Dashboard penjualan, laporan keuangan
  • Teknik: Aggregation, Data mining, Visualization
2. Diagnostic Analytics (Mengapa terjadi?)

Mengidentifikasi penyebab dari suatu kejadian

  • Contoh: Analisis penurunan penjualan
  • Teknik: Drill-down, Data discovery, Correlations
3. Predictive Analytics (Apa yang akan terjadi?)

Memprediksi kejadian di masa depan berdasarkan pola historis

  • Contoh: Forecasting demand, Credit scoring
  • Teknik: Machine learning, Regression, Time series
4. Prescriptive Analytics (Apa yang harus dilakukan?)

Memberikan rekomendasi tindakan optimal

  • Contoh: Optimization routes, Recommendation systems
  • Teknik: Optimization, Simulation, Decision analysis

Analytics Maturity Model

\[\text{Descriptive} \rightarrow \text{Diagnostic} \rightarrow \text{Predictive} \rightarrow \text{Prescriptive}\]

CPMK 043
Sub-CPMK 1.4

Pengertian Eksplorasi Data

Exploratory Data Analysis (EDA) adalah proses investigasi awal pada data untuk menemukan pola, anomali, menguji hipotesis, dan memeriksa asumsi menggunakan statistik ringkasan dan visualisasi.

Tujuan Eksplorasi Data

  • Memahami struktur dan karakteristik data
  • Mengidentifikasi missing values dan outliers
  • Menemukan pola dan hubungan antar variabel
  • Menentukan transformasi data yang diperlukan
  • Memvalidasi asumsi statistik

Aktivitas Eksplorasi Data

  1. Univariate Analysis
    • Distribusi single variable
    • Statistik deskriptif: mean, median, mode, std dev
    • Visualisasi: histogram, box plot
  2. Bivariate Analysis
    • Hubungan antara 2 variabel
    • Correlation analysis
    • Visualisasi: scatter plot, heat map
  3. Multivariate Analysis
    • Hubungan antar multiple variabel
    • Dimensionality reduction (PCA)
    • Clustering patterns

Korelasi Pearson

\[r = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n}(x_i - \bar{x})^2}\sqrt{\sum_{i=1}^{n}(y_i - \bar{y})^2}}\]

dimana \(r\) = koefisien korelasi, \(-1 \leq r \leq 1\)

Contoh: Correlation Heatmap

Metode: Ceramah, Praktek Studi Kasus
Penilaian: Tugas analisis dataset, Diskusi (5%)

Modelling: Regresi & Klasifikasi

CPMK 061
Sub-CPMK 2.3

Kemampuan Akhir yang Diharapkan

Mahasiswa mampu menjelaskan tahapan modelling Regresi dan Klasifikasi serta evaluasi model

Regresi (Regression)

Tujuan: Memprediksi nilai numerik kontinyu berdasarkan input features

Jenis-jenis Regresi

Linear Regression

Simple & Multiple Linear Regression

\[y = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_nx_n + \epsilon\]
Polynomial Regression

Non-linear relationships

\[y = \beta_0 + \beta_1x + \beta_2x^2 + ... + \beta_nx^n\]
Ridge & Lasso Regression

Regularized regression untuk prevent overfitting

Metrik Evaluasi Regresi

Mean Absolute Error (MAE): \[MAE = \frac{1}{n}\sum_{i=1}^{n}|y_i - \hat{y}_i|\] Mean Squared Error (MSE): \[MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2\] Root Mean Squared Error (RMSE): \[RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}\] R-Squared (R²): \[R^2 = 1 - \frac{\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{n}(y_i - \bar{y})^2}\]

Klasifikasi (Classification)

Tujuan: Memprediksi kategori/class label berdasarkan input features

Algoritma Klasifikasi

Logistic Regression

Binary & multiclass classification

Decision Tree

Tree-based classification

Random Forest

Ensemble of decision trees

Naive Bayes

Probabilistic classifier

K-Nearest Neighbors (KNN)

Instance-based learning

Support Vector Machine (SVM)

Maximum margin classifier

Metrik Evaluasi Klasifikasi

Accuracy: \[Accuracy = \frac{TP + TN}{TP + TN + FP + FN}\] Precision: \[Precision = \frac{TP}{TP + FP}\] Recall (Sensitivity): \[Recall = \frac{TP}{TP + FN}\] F1-Score: \[F1 = 2 \times \frac{Precision \times Recall}{Precision + Recall}\]

Modelling: Clustering

CPMK 061
Sub-CPMK 2.3

Kemampuan Akhir yang Diharapkan

Mahasiswa mampu menjelaskan tahapan modelling Clustering dan evaluasi model

Clustering (Unsupervised Learning)

Tujuan: Mengelompokkan data points yang similar ke dalam clusters tanpa predefined labels

Algoritma Clustering

1. K-Means Clustering
  • Partitioning method
  • Membagi data ke dalam K clusters
  • Minimize within-cluster variance
  • Fast & scalable
Objective Function: \[J = \sum_{i=1}^{K}\sum_{x \in C_i}||x - \mu_i||^2\]

dimana \(\mu_i\) = centroid of cluster \(C_i\)

2. Hierarchical Clustering
  • Agglomerative (bottom-up) atau Divisive (top-down)
  • Menghasilkan dendrogram
  • Tidak perlu specify jumlah clusters di awal
  • Linkage methods: single, complete, average, Ward
3. DBSCAN (Density-Based)
  • Density-based spatial clustering
  • Dapat menemukan clusters dengan arbitrary shapes
  • Robust terhadap outliers
  • Parameters: epsilon (ε), minPts
4. Gaussian Mixture Models (GMM)
  • Probabilistic clustering
  • Soft clustering (membership probabilities)
  • Assumes data dari mixture of Gaussian distributions

Metrik Evaluasi Clustering

Silhouette Score: \[s(i) = \frac{b(i) - a(i)}{\max\{a(i), b(i)\}}\]

dimana \(a(i)\) = avg distance to points in same cluster
\(b(i)\) = avg distance to points in nearest cluster
Range: [-1, 1], higher is better

Davies-Bouldin Index: \[DB = \frac{1}{K}\sum_{i=1}^{K}\max_{j \neq i}\left(\frac{\sigma_i + \sigma_j}{d(c_i, c_j)}\right)\]

Lower is better

Calinski-Harabasz Index (Variance Ratio): \[CH = \frac{SSB/(K-1)}{SSW/(N-K)}\]

Higher is better

Determining Optimal K

  • Elbow Method: Plot inertia vs K, find "elbow"
  • Silhouette Analysis: Choose K with highest avg silhouette score
  • Gap Statistic: Compare with null reference distribution

Contoh: Elbow Method

Tools

Python (scikit-learn) RapidMiner R

Confusion Matrix

Matrix yang menampilkan performa classification model dengan membandingkan predicted vs actual labels

Confusion Matrix Predicted
Positive Negative
Actual Positive True Positive (TP) False Negative (FN)
Type II Error
Negative False Positive (FP)
Type I Error		 True Negative (TN)

Metrik dari Confusion Matrix

1. Accuracy (Akurasi): \[Accuracy = \frac{TP + TN}{TP + TN + FP + FN}\]

Proporsi prediksi yang benar dari total prediksi

2. Precision (Presisi): \[Precision = \frac{TP}{TP + FP}\]

Dari yang diprediksi positif, berapa yang benar-benar positif?

3. Recall / Sensitivity / True Positive Rate: \[Recall = \frac{TP}{TP + FN}\]

Dari yang sebenarnya positif, berapa yang berhasil dideteksi?

4. Specificity / True Negative Rate: \[Specificity = \frac{TN}{TN + FP}\]

Dari yang sebenarnya negatif, berapa yang benar diprediksi negatif?

5. F1-Score (Harmonic Mean of Precision & Recall): \[F1 = 2 \times \frac{Precision \times Recall}{Precision + Recall}\]

Balance antara precision dan recall

6. F-Beta Score: \[F_\beta = (1 + \beta^2) \times \frac{Precision \times Recall}{\beta^2 \times Precision + Recall}\]

\(\beta > 1\): lebih pentingkan recall
\(\beta < 1\): lebih pentingkan precision

ROC Curve & AUC

  • ROC (Receiver Operating Characteristic): Plot TPR vs FPR pada berbagai threshold
  • AUC (Area Under Curve): Luas area di bawah ROC curve
    • AUC = 1.0: Perfect classifier
    • AUC = 0.5: Random classifier
    • AUC > 0.8: Good model

Regression Metrics (Review)

Mean Absolute Error (MAE): \[MAE = \frac{1}{n}\sum_{i=1}^{n}|y_i - \hat{y}_i|\]

Average absolute difference, less sensitive to outliers

Mean Squared Error (MSE): \[MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2\]

Penalizes larger errors more heavily

Root Mean Squared Error (RMSE): \[RMSE = \sqrt{MSE}\]

Same unit as target variable

Mean Absolute Percentage Error (MAPE): \[MAPE = \frac{100\%}{n}\sum_{i=1}^{n}\left|\frac{y_i - \hat{y}_i}{y_i}\right|\]

Percentage error, interpretable

R-Squared (Coefficient of Determination): \[R^2 = 1 - \frac{SS_{res}}{SS_{tot}} = 1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}\]

Proportion of variance explained, Range: [0, 1]

Adjusted R-Squared: \[R^2_{adj} = 1 - \frac{(1-R^2)(n-1)}{n-p-1}\]

Adjusted for number of predictors \(p\)

Cross-Validation

  • K-Fold Cross-Validation: Split data ke K folds, rotate train/test
  • Stratified K-Fold: Maintain class distribution in each fold
  • Leave-One-Out (LOO): K = n, untuk dataset kecil

Mini Project: Presentasi

CPMK 043
CPMK 061

Kemampuan Akhir yang Diharapkan

Mahasiswa mampu mempresentasikan hasil project dan menyajikan hasil analisis

Struktur Presentasi

1. Introduction (5 menit)
  • Background & business context
  • Problem statement
  • Objectives & success criteria
2. Data Overview (3 menit)
  • Dataset description
  • Data sources
  • Key features/variables
3. Methodology (5 menit)
  • Data preparation process
  • Analytical approach
  • Tools & techniques used
  • Challenges encountered & how overcome
4. Results & Findings (10 menit)
  • Exploratory Analysis: key patterns, distributions, correlations
  • Model Performance: evaluation metrics, comparison
  • Visualizations: charts, dashboards, graphs
  • Key Insights: main discoveries
5. Business Implications (5 menit)
  • How findings answer business questions
  • Actionable recommendations
  • Expected business impact
6. Conclusion & Future Work (2 menit)
  • Summary of key takeaways
  • Limitations
  • Future improvements atau extensions
7. Q&A (5-10 menit)
  • Respond to questions
  • Clarifications

Deliverables

Presentation Slides

PowerPoint/PDF dengan visualisasi yang jelas

Report

Written report dengan detail analisis

Code/Models

Script/workflow yang dapat direproduce

Dashboard (Optional)

Interactive dashboard jika applicable

Tips Presentasi Efektif:

  • Start strong dengan compelling story
  • Use visualizations effectively - "show, don't just tell"
  • Focus on insights, bukan hanya hasil teknis
  • Relate findings ke business impact
  • Practice timing - jangan terlalu cepat atau lambat
  • Prepare untuk anticipated questions
  • Be confident dan enthusiastic!
